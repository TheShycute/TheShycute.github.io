## 🚀 本地运行

### 环境要求

- [Ruby](https://www.ruby-lang.org/) 3.0+
- [Bundler](https://bundler.io/)

### 安装运行

```bash
# 克隆仓库
git clone https://github.com/TheShycute/TheShycute.github.io.git
cd TheShycute.github.io

# 安装依赖
bundle install

# 启动本地服务（默认 localhost:4000）
bundle exec jekyll serve
```

## 📝 撰写文章

在 `_posts/` 目录下新建 Markdown 文件，命名格式为 `YYYY-MM-DD-title.md`。

文章头部需要 Front Matter：

```yaml
---
layout: post
title: "文章标题"
subtitle: "副标题（可选）"
date: 2026-05-31 12:00:00
author: "天使皮卡丘丨"
header-img: "img/post-bg-xxx.jpg"
catalog: true
tags:
  - 标签1
  - 标签2
---
```

推送后 GitHub Actions 会自动构建部署，1-3 分钟内生效。
