## 学习路线

黑马程序员是我系统学习 Java 全栈开发的重要起点。课程涵盖了从基础到进阶的完整知识体系。

## 课程体系

### Java 基础
- 面向对象编程（封装、继承、多态）
- 集合框架（List、Set、Map）
- IO 流与网络编程
- 多线程与并发

### 数据库
- MySQL 基础与进阶
- JDBC 数据库连接
- MyBatis 持久层框架
- Redis 缓存

### 前端基础
- HTML5 + CSS3
- JavaScript ES6+
- Vue.js 基础

### Spring 全家桶
- Spring Framework（IoC、AOP）
- Spring MVC
- Spring Boot 自动配置
- Spring Security 安全框架

### 项目实战
- 在线教育平台（学成在线）
- 苍穹外卖
- 微服务架构入门

## 学习心得

> "编程不是学会的，是做会的。"

黑马的课程最大的特点就是**项目驱动**——每个阶段都有对应的实战项目。通过动手实践，抽象的概念变得具体可感。
