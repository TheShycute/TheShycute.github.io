## 什么是计算机视觉

计算机视觉（Computer Vision，CV）是让机器"看懂"图像和视频的技术领域。从简单的图像分类到复杂的场景理解，CV 技术已经渗透到安防、医疗、自动驾驶、工业检测等方方面面。

## CV 技术金字塔

```
           ┌─────────────────┐
           │   场景理解 / VQA  │  ← 最高层
           ├─────────────────┤
           │  视频分析 / 跟踪  │
           ├─────────────────┤
           │ 目标检测 / 分割   │
           ├─────────────────┤
           │ 图像分类 / 识别   │
           ├─────────────────┤
           │ 图像处理 / 增强   │  ← 基础层
           └─────────────────┘
```

## 一、图像处理基础

### 核心操作

| 操作 | 作用 | OpenCV 函数 |
|------|------|------------|
| 灰度化 | 降低计算量 | `cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)` |
| 高斯模糊 | 降噪平滑 | `cv2.GaussianBlur(img, (5,5), 0)` |
| 边缘检测 | 提取轮廓 | `cv2.Canny(img, 50, 150)` |
| 二值化 | 前景/背景分离 | `cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)` |
| 形态学操作 | 膨胀/腐蚀/开闭运算 | `cv2.dilate()` / `cv2.erode()` |
| 直方图均衡化 | 增强对比度 | `cv2.equalizeHist(img)` |

### 颜色空间

```
RGB     → 显示设备原生，三通道
HSV     → 色调/饱和度/明度，更适合颜色筛选
Lab     → 感知均匀，适合颜色距离计算
YCrCb   → 亮度和色度分离，肤色检测常用
灰度    → 单通道，简化计算
```

### 图像滤波

| 滤波器 | 效果 | 适用场景 |
|--------|------|---------|
| **均值滤波** | 平滑但模糊边缘 | 基础降噪 |
| **高斯滤波** | 保留更多边缘信息 | 预处理首选 |
| **中值滤波** | 去除椒盐噪声 | 传感器噪声 |
| **双边滤波** | 保边去噪 | 美颜、细节保留 |

## 二、传统 CV 技术

### 特征提取

```
SIFT（尺度不变特征变换）
- 关键点检测 + 方向赋值 + 描述子生成
- 尺度不变、旋转不变
- 专利保护，OpenCV 4.4+ 需单独编译

ORB（快速二值描述子）
- 基于 FAST 角点 + BRIEF 描述子
- 速度快，免费开源
- 实时应用首选

HOG（方向梯度直方图）
- 统计局部梯度方向
- 行人检测经典方案
- 配合 SVM 分类器
```

### 图像配准与拼接

```
图像 A ───┐
          ├─ 特征匹配 → 变换矩阵 → 融合 → 全景图
图像 B ───┘
```

## 三、卷积神经网络（CNN）

### CNN 核心组件

```
输入图像 (224×224×3)
    │
    ▼
┌────────────────────┐
│   卷积层 (Conv)      │  ← 提取特征（边缘→纹理→语义）
│   激活函数 (ReLU)    │  ← 引入非线性
│   池化层 (Pooling)   │  ← 降维、防过拟合
└────────────────────┘
    │  （重复多次）
    ▼
┌────────────────────┐
│   全连接层 (FC)      │  ← 分类/回归
└────────────────────┘
```

### 经典 CNN 架构演化

| 模型 | 年份 | 核心创新 | Top-5 错误率 |
|------|------|---------|-------------|
| LeNet-5 | 1998 | CNN 鼻祖，手写数字识别 | - |
| AlexNet | 2012 | ReLU、Dropout、GPU | 15.3% |
| VGG-16 | 2014 | 小卷积核 (3×3) 堆叠 | 7.3% |
| GoogLeNet | 2014 | Inception 多分支结构 | 6.7% |
| ResNet-50 | 2015 | 残差连接，解决退化 | 3.6% |
| DenseNet | 2017 | 密集连接 | 3.5% |
| EfficientNet | 2019 | 复合缩放 | 2.9% |

### ResNet 残差连接原理

```
传统网络：              残差网络：
x → Conv → F(x)        x → Conv → F(x)
                              ↓
                         x ────→ F(x) + x  ← 残差连接

优势：梯度可以直接"跳过"层传递，避免梯度消失
```

## 四、目标检测模型

| 类型 | 代表模型 | 特点 |
|------|---------|------|
| **两阶段** | Faster R-CNN | 高精度，速度慢 |

## 五、图像分割

### 语义分割 vs 实例分割 vs 全景分割

```
原图（两人+一狗）
    │
    ├─ 语义分割 → 所有人标注为"人"，所有像素同一颜色
    │
    ├─ 实例分割 → 人1、人2、狗分别不同颜色 + 独立 mask
    │
    └─ 全景分割 → 语义分割（背景）+ 实例分割（前景）统一
```

### 经典分割模型

| 模型 | 类型 | 关键设计 |
|------|------|---------|
| FCN | 语义分割 | 全卷积，端到端 |
| U-Net | 语义分割 | 编码-解码 + 跳跃连接 |
| Mask R-CNN | 实例分割 | 在 Faster R-CNN 上加 mask 分支 |
| YOLOv8-seg | 实例分割 | 一体化检测+分割 |

## 六、生成式模型与更多前沿

### GAN（生成对抗网络）

```
生成器 G：随机噪声 → 伪造图像
判别器 D：判断图像真假

训练过程：G 和 D 相互博弈，G 越来越会"造假"，D 越来越会"识假"
```

### 扩散模型（Diffusion Models）

Stable Diffusion、DALL-E 的核心技术：
- 正向过程：逐步给图像加噪
- 反向过程：从噪声中逐步还原图像
- 文本引导：用文本 embedding 控制生成方向

### SAM（Segment Anything Model）

Meta 发布的通用分割大模型：
- 点一下，分割一个物体
- 框一下，分割框内物体
- 全自动分割整张图

```

```
