## 项目简介

金属细管广泛应用于航空航天、汽车制造等领域，其内壁质量直接关系到产品安全。本系统基于 YOLO 深度学习模型，实现了金属细管内壁缺陷的自动化检测。

## 检测模式

系统支持四种灵活的检测模式：

| 模式 | 说明 |
|------|------|
| 📷 单张图片 | 上传单张图片进行检测 |
| 📁 批量图片 | 批量上传多张图片，自动处理 |
| 🎥 摄像头实时 | 连接摄像头进行实时检测 |
| 📹 视频文件 | 上传视频文件进行逐帧检测 |

## 技术栈

| 层 | 技术 |
|---|---|
| 前端框架 | React 18 + TypeScript |
| 构建工具 | Webpack 5 |
| 样式方案 | Tailwind CSS + CSS 自定义属性 |
| 深度学习 | YOLO 模型 |

## 技术挑战

1. **小目标检测**：金属管内壁缺陷通常较小，需要针对性的数据增强和模型调优
2. **实时性能**：摄像头模式下需要保证低延迟，涉及模型量化和推理优化
3. **多模式适配**：四种检测模式的统一架构设计

## 项目收获

通过这个项目，我深入掌握了 YOLO 模型的训练、调优和部署流程，以及 React + TypeScript 在中大型前端项目中的工程化实践。
